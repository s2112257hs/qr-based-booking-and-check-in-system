This project uses Next.js Pages Router.

Route files live directly under `src/pages`, for example:
- `src/pages/index.tsx`
- `src/pages/login.tsx`
- `src/pages/trips.tsx`
- `src/pages/bookings.tsx`
- `src/pages/scan.tsx`
- `src/pages/admin/*`

Shared UI is in `src/components` and route guards are in `src/hooks`.
